## ListFooter
A special title block that can be placed at the bottom of a List

#### Examples:

```javascript
import {List, ListFooter} from 'gadgets';
<List>
    <ListFooter title"footer string" />
</List>
```

#### Events
See `Title` component

#### Styles
- `ui-list-footer` - A class style on the title block of the
component.

#### Properties
See `Title` component
